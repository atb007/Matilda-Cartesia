## What's new in v1.0.4

### DAW Sync UI
- **Top-right canvas toggle** — DAW Sync moved out of Global Settings to a hero-frame control (mirrors the collapse chevron), using `DawSyncOn` / `DawSyncOff` assets. Default **On**.
- **Linked play gem** — When sync is on, the play button shows a link icon instead of play/stop; manual transport from the plugin is disabled so the host drives start/stop.
- **Hover tips** — Collapse chevron and DAW sync toggle show delayed tooltips on hover.
- **Press feedback** — Subtle scale-down click animation on collapse, sync, and play gem buttons; pointer cursor on clickable controls.

### Polish
- Link icon aspect ratio and size tuned (no horizontal stretch).
- DAW sync toggle visible in standalone for UI preview.
- Hero subtitle shows **Cartesia - v1.0.4** (from `matilda/plugin/VERSION`).

## Install

### macOS
- **VST3:** unzip `Matilda-macOS-vst3.zip` → copy `Matilda.vst3` to `~/Library/Audio/Plug-Ins/VST3/`
- **Standalone:** unzip `Matilda-macOS-standalone.zip` → open `Matilda.app`
- Universal binary (Apple Silicon + Intel)

### Windows (FL Studio, etc.)
- **VST3:** unzip `Matilda-Windows-vst3.zip` → copy `Matilda.vst3` to `C:\Program Files\Common Files\VST3\`
- **Standalone:** unzip `Matilda-Windows-standalone.zip` → run `Matilda.exe`
- Rescan plugins in your DAW after copying the VST3 folder

Matilda is a **MIDI arpeggiator** — pair it with a synth on the same channel (or route its MIDI out) to hear the pattern.
