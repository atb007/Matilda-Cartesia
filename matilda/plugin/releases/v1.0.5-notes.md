## What's new in v1.0.5

### FL Studio / DAW MIDI routing
- **VST3 MIDI effect** — Matilda registers as a MIDI-FX (BlueARP-style), not a silent synth instrument. Pair with a synth in Fruity Wrapper or Patcher.
- **Auto-arm with DAW Sync** — Sequencer arms on load when DAW Sync is on (default); follows host transport without pressing the play gem.
- **MIDI transport fallback** — Responds to MIDI Start/Stop/Clock when the host playhead is unavailable (common in FL Fruity Wrapper).

### UI fix (Windows VST)
- **Filigree ornaments** — Title decorations (Quantise Scale, Global Settings, Movement) render muted silver instead of black on Windows; cross-platform pixel tint fix.

## Install

### macOS
- **VST3:** unzip `Matilda-macOS-vst3.zip` → copy `Matilda.vst3` to `~/Library/Audio/Plug-Ins/VST3/`
- **Standalone:** unzip `Matilda-macOS-standalone.zip` → open `Matilda.app`
- Universal binary (Apple Silicon + Intel)

### Windows (FL Studio, etc.)
- **VST3:** unzip `Matilda-Windows-vst3.zip` → copy `Matilda.vst3` to `C:\Program Files\Common Files\VST3\`
- **Standalone:** unzip `Matilda-Windows-standalone.zip` → run `Matilda.exe`
- Rescan plugins in your DAW after copying the VST3 folder

### FL Studio Fruity Wrapper
1. Matilda in wrapper slot 1, synth in slot 2.
2. Wire Matilda **MIDI output port N** → synth **MIDI input port N** (ports 11+).
3. Press Play on FL transport (DAW Sync on by default).

Matilda is a **MIDI arpeggiator** — pair it with a synth to hear the pattern.
