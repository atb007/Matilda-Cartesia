## What's new in v1.0.6

### FL Studio / Fruity Wrapper — sound fix
- **Opaque-host transport fallback** — When DAW Sync is on but FL Wrapper omits playhead and MIDI Start/Clock (common), Matilda steps on the **internal sample clock** instead of staying silent.
- **Auto-start when armed** — Sequencer begins stepping immediately in opaque-host mode; no longer blocked waiting for undetectable host transport.

### UI — filigree parity with Standalone / React
- **Native SVG drawables** — Quantise Scale, Global Settings, and Movement titles use the same panel-specific SVG assets as the React shell (no Windows raster-tint pass that rendered black vines).
- **Correct assets per panel** — `scale-title-filigree`, `transport-title-filigree`, `movement-filigree` (was incorrectly reusing movement SVG everywhere).

### UI — layout persistence
- **Remember window scale** — User resize (70%–100%) saved in plugin state; restored when reopening in FL or other DAWs.
- **Remember collapse state** — Expanded/collapsed shell preference saved across sessions.

## Install

### macOS
- **VST3:** unzip `Matilda-macOS-vst3.zip` → copy `Matilda.vst3` to `~/Library/Audio/Plug-Ins/VST3/`
- **Standalone:** unzip `Matilda-macOS-standalone.zip` → open `Matilda.app`
- Universal binary (Apple Silicon + Intel)

### Windows (FL Studio, etc.)
- **VST3:** unzip `Matilda-Windows-vst3.zip` → copy `Matilda.vst3` to `C:\Program Files\Common Files\VST3\`
- **Standalone:** unzip `Matilda-Windows-standalone.zip` → run `Matilda.exe`
- Rescan plugins in your DAW after copying the VST3 folder

### FL Studio Fruity Wrapper
1. Fruity Wrapper on channel rack (BlueARP-style “instrument”).
2. Matilda in wrapper slot 1, synth (3xOsc, Sytrus, etc.) in slot 2.
3. Wire Matilda **MIDI output port N** → synth **MIDI input port N** (any matching number).
4. DAW Sync **On** (default) → press **Play** on FL transport.
5. Playhead should move and synth should play within ~1 s — no piano-roll MIDI required.

Matilda is a **MIDI arpeggiator** — pair it with a synth to hear the pattern.
