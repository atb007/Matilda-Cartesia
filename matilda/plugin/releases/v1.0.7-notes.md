## What's new in v1.0.7

### UI — filigree orientation fixes
- **Bottom title filigree** — Movement and Quantise Scale bottom vines now mirror vertically (Figma `scaleY(-1)`), matching Global Settings and the React shell (were drawn un-flipped / rotated).
- **Section ornaments** — Play Mode and Clock **right-side** ornaments now mirror horizontally (Figma `scaleX(-1)`) so they point inward toward the label, matching the left ornament.

> Known limitation: in FL Studio, route Matilda's MIDI to a synth requires a **VST2** host path. FL does not forward MIDI output from VST3 plugins to other plugins (host limitation, not a Matilda bug). See release discussion.

## Install

### macOS
- **VST3:** unzip `Matilda-macOS-vst3.zip` → copy `Matilda.vst3` to `~/Library/Audio/Plug-Ins/VST3/`
- **Standalone:** unzip `Matilda-macOS-standalone.zip` → open `Matilda.app`

### Windows (FL Studio, etc.)
- **VST3:** unzip `Matilda-Windows-vst3.zip` → copy `Matilda.vst3` to `C:\Program Files\Common Files\VST3\`
- **Standalone:** unzip `Matilda-Windows-standalone.zip` → run `Matilda.exe`
- Rescan plugins after copying the VST3 folder.
